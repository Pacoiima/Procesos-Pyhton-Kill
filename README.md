# Procesos-Pyhton-Kill
Automatización en python para procesos que superen el limite de memoria RAM.

Con la libreria "paramiko" ingresamos a maquina virtual por medio de ssh. Esto con el fin de monitorear los procesos que se lleven acabo en dicha maquina y poder elimiar procesos que superen el limite de memoria RAM.

Aunque esta hecho con una consulta para procesar memoria RAM, tambien se puede añadir a la consulta un procedimiento para el limite de CPU.
